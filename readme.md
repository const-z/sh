SmartHome - библиотека, которая позволяет моделировать умный дом.

## Бибилиотека умной розетки

`c_socket_lib` - библиотека экспортирует C-style функции управления розеткой.  
`c_socket_use_static` - приложение импортирует статическую библиотеку c_socket_lib и демонстрирует управление розеткой через предоставленные ей функции.  
`c_socket_use_runtime` - приложение импортирует динамическую библиотеку c_socket_lib (в runtime) и демонстрирует управление розеткой через предоставленные ей функции.  

`cargo test` - выполнит тесты
